# Filippa Colbin's Java Projects

"This repository contains Java projects from my studies, showcasing my programming skills."

## Folder Structure

- **calculations/**  
  - `SolarPanelProductionCalculator.java` – Calculates solar panel electricity production.  
  - `RandomNumbersEvenOdd.java` – Generates random numbers and sorts them into even and odd arrays.  
  - `AreaVolumeFractionCalculator.java` – Calculates areas, volumes, and simplifies fractions.  

- **games/**  
  - `DiceGame12.java` – Dice game where the goal is to roll a sum of 12.  

- **file-handling/**  
  - `CashRegisterSystem.java` – Manages items, sales, and maintains a sales history.  
  - `LibrarySystem.java` – Simple library system for adding, removing, loaning, and returning books.
